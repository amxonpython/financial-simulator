# 💰 Финансовая симуляция

Консольное Java-приложение для симуляции личных финансов: учёт баланса, доходов и расходов, расчёт налога по прогрессивной шкале НДФЛ и моделирование дохода по вкладу.

## Возможности

- **Ручной режим** — пошаговое управление балансом через консольное меню (пополнение, списание, симуляция месяца, просмотр прибыли).
- **Автоматический режим** — симуляция сразу на 12 месяцев (год) без ручного ввода на каждом шаге.
- **Расчёт налога** — прогрессивная шкала ставки НДФЛ (13% / 15% / 18% / 20% / 22%) в зависимости от суммы дохода, с необлагаемым лимитом.
- **Доход по вкладу** — расчёт ежемесячного дохода от размещения баланса на вкладе по заданной годовой ставке (по умолчанию 9%).
- **Расчёт прибыли** — прибыль считается как доход минус расходы минус налог.

## Структура проекта

```
├── Data/
│   ├── Data.java            # Базовые данные: баланс, доход, расходы, прибыль
│   ├── tax.java             # Прогрессивный расчёт налога (НДФЛ)
│   └── contribution.java    # Расчёт дохода по вкладу
├── entryPoint/
│   ├── Main.java                    # Точка входа, обработка команд
│   ├── ManagementViaTheConsole.java # Меню ручного управления
│   ├── help/
│   │   └── help.java                # Список доступных команд
│   └── AutoSimulation/
│       └── simulation.java          # Автоматическая симуляция на N месяцев
├── financialActions/
│   └── actions.java          # Операции над балансом (пополнение, списание и т.д.)
└── sysData/
    └── actions.java          # Вывод меню действий в консоли
```

## Как запустить

Требуется JDK 17+ (используется `switch` с несколькими метками в одном `case`).

```bash
javac -d out $(find . -name "*.java")
java -cp out entryPoint.Main
```

## Команды в главном меню

| Команда | Описание |
|---|---|
| `help` | показать список доступных команд |
| `autoSim` / `run the simulation: auto` | запустить автоматическую симуляцию на 12 месяцев |
| `sim` / `run the simulation: not auto` | запустить ручной режим управления балансом |
| `exit` | выйти из программы |

## Действия в ручном режиме

| № | Действие |
|---|---|
| 0 | показать баланс |
| 1 | пополнить баланс |
| 2 | уменьшить баланс |
| 3 | симулировать изменение баланса за 1 месяц |
| 4 | рассчитать общую прибыль |
| 5 | выход |

## Статус проекта

Учебный/пет-проект. Возможны шероховатости в логике расчётов и незавершённые части кода — пул-реквесты и предложения приветствуются.
